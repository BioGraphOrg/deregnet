Detailed Description

template<typename ModelType, typename Data>
class deregnet::DeregnetFinder< ModelType, Data >

Main entry point for running DeRegNet algorithms.

DeregnetFinder has two template paramters: 1) ModelType, referring to either GRBModel or grbfrc::FMILP, 2) Data, referring to a type representing algorithm data suitable for the ModelType parameter. In case of ModelType = GRBModel, Data should be DrgntData, in case of ModelType = grbfrc::FMILP, Data should be AvgdrgntData.

Todo:
This desgin seems not very optimal and somehow flawed. The most straight-forward improvment would be to implement the absolute version of the algorithm, i.e. ModelType = GRBModel, in terms of grbfrc::FMILP. ModelType would go away and Data could specfiy which algorithm to run, for example.
