Detailed Description

Base lazy constraint callback class.

Implements most of the basic logic concerning lazy constraints. Specializations arise from whether there is a fixed root or not (see LazyConstraintCallbackRoot and LazyConstraintCallbackNoRoot) and/or whether variable groups have to be mapped with respect to reformulations of the original problem (see GrbfrcLazyConstraintCallbackRoot and GrbfrcLazyConstraintCallbackNoRoot).
